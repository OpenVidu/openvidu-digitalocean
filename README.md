# openvidu-digitalocean
Terraform files to deploy all the types of deployments of OpenVidu in Digital Ocean cloud. You can check the documentation for the deployments in the following links: 
1. [Single Node Community deployment](https://openvidu.io/latest/docs/self-hosting/single-node/digitalocean/install){target="_blank"}
2. [Single Node PRO deployment](https://openvidu.io/latest/docs/self-hosting/single-node-pro/digitalocean/install){target="_blank"}
3. [Elastic deployment](https://openvidu.io/latest/docs/self-hosting/elastic/digitalocean/install){target="_blank"}
4. [High Availability deployment](https://openvidu.io/latest/docs/self-hosting/ha/digitalocean/install){target="_blank"}
